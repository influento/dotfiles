# ManimCE Animation Skill

## Workflow: Plan → Code → Render → Iterate

### Phase 0: Environment Check

Before first use, verify dependencies:

```bash
manim checkhealth        # Checks ManimCE + ffmpeg + LaTeX + Cairo
ffmpeg -version          # Needed for stitching scenes
```

If manim is not installed, set up a venv:

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install manim
```

### Phase 1: Plan

Create a dedicated project folder and write `plan.md` before any code:

```bash
mkdir -p <project-name> && cd <project-name>
```

```
<project-name>/
├── plan.md              # Planning document
├── script.py            # Manim code
├── manim.cfg            # Project-specific config (optional)
├── concat.txt           # ffmpeg scene list
├── final.mp4            # Stitched output
└── media/               # Auto-generated by manim
    └── videos/script/<quality>/
```

#### plan.md Template

```markdown
# [Video Title]

- **Topic**: [Core concept]
- **Hook**: [Opening question/mystery]
- **Target Audience**: [Prerequisites]
```

#### Narrative Patterns (pick one or combine)

1. **Mystery → Investigation → Resolution** — Present a puzzle, explore visually, reveal the principle
2. **Build Up → Payoff** — Simple blocks → complex result → reflect
3. **Two Perspectives → Unity** — Show algebraic + geometric views, reveal they're the same
4. **Wrong → Less Wrong → Right** — Naive approach fails → refine → correct understanding
5. **Specific → General** — Concrete example → notice pattern → abstract to principle
6. **History as Narrative** — Present problem historically, follow the discovery

### Phase 2: Code

Write `script.py` with one class per scene.

#### Code Structure Rules

1. **One class per scene** — Named: `Scene1_Introduction`, `Scene2_FindMax`
2. **Always add subtitles** via `add_subcaption()` — auto-generates `.srt`
3. **Use `from manim import *`** — ManimCE, not ManimGL

```python
from manim import *

class Scene1_Introduction(Scene):
    def construct(self):
        title = Text("Finding the Maximum", font_size=48, color=BLUE)
        self.add_subcaption("Let's find the max element", duration=2)
        self.play(Write(title))
        self.wait(1)
        self.play(FadeOut(title))
```

### Phase 3: Render

```bash
# Render one or more scenes (low quality for iteration)
manim -ql script.py Scene1_Introduction Scene2_FindMax

# Quality flags:
#   -ql  480p15   (fast, default for development)
#   -qm  720p30   (medium)
#   -qh  1080p60  (high, for final output)
#   -qp  1440p60  (2K)
#   -qk  2160p60  (4K)
```

Output: `media/videos/script/<quality>/SceneName.mp4`

**If a scene fails:** Read the error, fix the code, re-render only the failed scene.

#### Stitch Scenes

```bash
cat > concat.txt << 'EOF'
file 'media/videos/script/480p15/Scene1_Introduction.mp4'
file 'media/videos/script/480p15/Scene2_FindMax.mp4'
EOF

ffmpeg -y -f concat -safe 0 -i concat.txt -c copy final.mp4
```

### Phase 4: Iterate

When user provides feedback:
1. Identify which scenes need changes
2. Modify only those scenes in `script.py`
3. Re-render only affected scenes
4. Re-stitch with ffmpeg
5. Show user the result

## ManimCE Best Practices Reference

### Scene Structure

```python
class MyScene(Scene):
    def construct(self):
        # All animation code goes here
        pass

# Scene types:
# Scene                      — standard 2D
# MovingCameraScene           — zoom, pan
# ThreeDScene                 — 3D with camera orientation
# ZoomedScene                 — magnification lens effect
# VectorScene                 — vector math
# LinearTransformationScene   — matrix transformation visualization
```

`setup()` runs before `construct()` — use for initialization:

```python
class MyScene(Scene):
    def setup(self):
        self.camera.background_color = "#1a1a2e"
    def construct(self):
        ...
```

Scene methods:
```python
self.add(mob)              # Instant add
self.remove(mob)           # Instant remove
self.play(Create(mob))     # Animated
self.play(A, B, C)         # Simultaneous
self.wait(2)               # Pause
```

#### Creation
```python
Create(shape)              # Draw progressively (shapes, lines)
Write(text)                # Handwriting effect (text, equations)
Unwrite(text)              # Reverse of Write
FadeIn(mob)                # Opacity transition
FadeIn(mob, shift=UP)      # Directional fade
DrawBorderThenFill(mob)    # Outline then fill
GrowFromCenter(mob)        # Grow from center
GrowFromEdge(mob, LEFT)    # Grow from edge
GrowArrow(arrow)           # Grow arrow from start to tip
SpinInFromNothing(mob)     # Spin + grow from center
SpiralIn(shapes)           # Multiple objects spiral in
AddTextLetterByLetter(text, time_per_char=0.1)  # Typing effect (Text only)
ShowIncreasingSubsets(group)  # Reveal submobjects cumulatively
```

#### Removal
```python
FadeOut(mob)               # Fade to transparent
FadeOut(mob, shift=DOWN)   # Directional fade out
Uncreate(mob)              # Reverse of Create
ShrinkToCenter(mob)        # Shrink and vanish
RemoveTextLetterByLetter(text)  # Reverse typing
```

#### Transform
```python
Transform(A, B)            # Morph A into B (A still referenced)
ReplacementTransform(A, B) # Replace A with B (B is now in scene)
TransformFromCopy(A, B)    # Keep A, create B from copy
TransformMatchingTex(eq1, eq2)   # Match LaTeX parts by tex string
TransformMatchingShapes(A, B)    # Match by shape similarity
FadeTransform(A, B)        # Cross-fade between mobjects
MoveToTarget(mob)          # Animate to pre-set target (needs generate_target())
Swap(A, B)                 # Swap positions of two mobjects
CyclicReplace(A, B, C)    # Cycle positions: A→B→C→A
ApplyMatrix(matrix, mob)   # Apply 2x2/3x3 matrix transformation
Restore(mob)               # Restore to saved state (needs save_state())
```

**Use ReplacementTransform for clarity** — more intuitive variable behavior.

**Path arcs for visual interest:**
```python
self.play(Transform(A, B, path_arc=PI/2))      # Curved path
self.play(ClockwiseTransform(A, B))             # Clockwise arc
self.play(CounterclockwiseTransform(A, B))      # Counter-clockwise
```

#### Groups & Sequencing
```python
# Simultaneous with stagger
self.play(LaggedStart(*[Create(c) for c in circles], lag_ratio=0.1))

# Sequential in one play call
self.play(Succession(Create(a), Create(b), Create(c)))

# Apply same animation to all submobjects
self.play(LaggedStartMap(FadeIn, group, lag_ratio=0.1))
```

`lag_ratio`: 0 = all simultaneous, 0.5 = each starts at 50%, 1 = sequential.

#### Attention & Indication
```python
Indicate(mob)              # Scale + recolor flash (uses there_and_back)
Circumscribe(mob)          # Draw shape around (Rectangle or Circle)
Flash(point)               # Radiating lines from point
Wiggle(mob)                # Oscillating wobble
ApplyWave(mob)             # Wave distortion (ripples=N for multiple)
FocusOn(point)             # Shrinking spotlight
ShowPassingFlash(mob)      # Traveling highlight along stroke
Blink(cursor, blinks=3)   # Blink on/off (for cursor effects)
```

#### Movement
```python
MoveAlongPath(mob, path)   # Follow a VMobject path
Rotating(mob, angle=TAU)   # Continuous rotation (default: full turn)
Homotopy(func, mob)        # Arbitrary point transformation f(x,y,z,t)
```

### Graph (Graph Theory — Nodes & Edges)

```python
# Undirected graph
g = Graph(
    vertices=[1, 2, 3, 4, 5],
    edges=[(1, 2), (2, 3), (3, 4), (4, 5), (1, 5)],
    layout="spring",     # Layout algorithm
    labels=True,         # Show vertex labels
)

# Directed graph
dg = DiGraph(
    vertices=[1, 2, 3],
    edges=[(1, 2), (2, 3), (3, 1)],
    layout="circular",
    labels=True,
    edge_config={"tip_config": {"tip_length": 0.2}},
)

# Layout algorithms:
#   "spring"       — force-directed (default, good general purpose)
#   "circular"     — vertices on circle
#   "kamada_kawai" — distance-preserving
#   "tree"         — hierarchical (set root_vertex in layout_config)
#   "planar"       — non-crossing edges
#   "partite"      — partition-based (set partitions in layout_config)
#   "spiral", "shell", "spectral", "random"

# Tree layout
tree = Graph(
    [0, 1, 2, 3, 4, 5, 6],
    [(0,1), (0,2), (1,3), (1,4), (2,5), (2,6)],
    layout="tree",
    layout_config={"root_vertex": 0},
    labels=True,
)

# Dynamic: add/remove vertices and edges
g.add_vertices(6, positions={6: RIGHT * 3})
g.add_edges((5, 6))
g.remove_edges((1, 2))
g.remove_vertices(3)

# Change layout
g.change_layout("circular")

# Style individual elements
g.vertices[1].set_color(RED)
g.edges[(1, 2)].set_color(YELLOW)
```

### Array Visualization

```python
def create_array(values, cell_width=1.0, cell_height=0.8, font_size=36):
    """Create a visual array with indexed cells."""
    cells = VGroup()
    for i, val in enumerate(values):
        rect = Rectangle(width=cell_width, height=cell_height,
                         stroke_color=WHITE, fill_color=BLUE_E, fill_opacity=0.3)
        text = Text(str(val), font_size=font_size)
        idx = Text(str(i), font_size=font_size * 0.6, color=GREY_B)
        text.move_to(rect)
        idx.next_to(rect, DOWN, buff=0.15)
        cell = VGroup(rect, text, idx)
        cells.add(cell)
    cells.arrange(RIGHT, buff=0)
    return cells
```

### Pointer / Highlight

```python
def create_pointer(array_cell, label_text, color=YELLOW, direction=UP):
    """Arrow pointing at an array cell with a label."""
    if direction is UP:
        start = array_cell.get_top() + UP * 0.8
        end = array_cell.get_top()
    else:
        start = array_cell.get_bottom() + DOWN * 0.8
        end = array_cell.get_bottom()
    arrow = Arrow(start=start, end=end, color=color, buff=0.1)
    label = Text(label_text, font_size=28, color=color)
    label.next_to(arrow, direction, buff=0.1)
    return VGroup(arrow, label)
```

### Step-by-Step Algorithm Execution

Pattern for animating algorithm iterations:

```python
class FindMax(Scene):
    def construct(self):
        # Title
        title = Text("Finding Maximum — O(n)", font_size=40)
        title.to_edge(UP)
        self.play(Write(title))

        # Array
        values = [3, 7, 2, 9, 4, 1, 8]
        array = create_array(values)
        array.move_to(ORIGIN)
        self.play(LaggedStart(*[FadeIn(c) for c in array], lag_ratio=0.1))

        # Variable display (built-in)
        max_var = Variable(values[0], Text("max", font_size=28), num_decimal_places=0)
        max_var.label.set_color(GREY_B)
        max_var.value.set_color(YELLOW)
        max_var.to_corner(UR)
        self.play(FadeIn(max_var))

        # Pointer
        pointer = create_pointer(array[0], "i", GREEN)
        self.play(Create(pointer))

        # Iterate
        current_max = values[0]
        for i in range(1, len(values)):
            # Move pointer
            new_pointer = create_pointer(array[i], "i", GREEN)
            self.play(ReplacementTransform(pointer, new_pointer), run_time=0.5)
            pointer = new_pointer

            # Highlight comparison
            self.play(array[i][0].animate.set_fill(YELLOW, opacity=0.5), run_time=0.3)

            if values[i] > current_max:
                current_max = values[i]
                self.play(Indicate(array[i][1], color=YELLOW))
                self.play(max_var.tracker.animate.set_value(current_max), run_time=0.3)

            # Reset cell color
            self.play(array[i][0].animate.set_fill(BLUE_E, opacity=0.3), run_time=0.2)

        # Highlight result
        max_idx = values.index(max(values))
        self.play(array[max_idx][0].animate.set_fill(GREEN, opacity=0.6))
        self.play(Circumscribe(array[max_idx], color=GREEN))
        self.wait(2)
```

### Memory Cell / RAM Model

```python
def create_memory_block(address, value, width=1.5, height=0.7):
    """Single memory cell with address label."""
    rect = Rectangle(width=width, height=height,
                     stroke_color=WHITE, fill_color=DARK_GREY, fill_opacity=0.5)
    addr = Text(f"0x{address:02X}", font_size=20, color=GREY_B)
    val = Text(str(value), font_size=28, color=WHITE)
    addr.next_to(rect, LEFT, buff=0.2)
    val.move_to(rect)
    return VGroup(addr, rect, val)

def create_memory_column(start_addr, values):
    """Vertical stack of memory cells."""
    cells = VGroup(*[
        create_memory_block(start_addr + i, v)
        for i, v in enumerate(values)
    ])
    cells.arrange(DOWN, buff=0)
    return cells
```

```python
# Binary tree using Graph with tree layout
def create_binary_tree(depth):
    """Create a binary tree Graph with given depth."""
    vertices = list(range(2**depth - 1))
    edges = [(i, 2*i+1) for i in range(len(vertices)//2)]
    edges += [(i, 2*i+2) for i in range(len(vertices)//2)]
    edges = [(u, v) for u, v in edges if v < len(vertices)]
    return Graph(
        vertices, edges,
        layout="tree", layout_config={"root_vertex": 0},
        labels=True,
    )
```

```python
def create_linked_list(values):
    """Horizontal linked list with arrows."""

    for val in values:
```

## Visual Design Principles

### Progressive Disclosure
Build complexity gradually. Don't show everything at once.

### Transform, Don't Replace
Use `TransformMatchingTex` / `ReplacementTransform` instead of `FadeOut` + `FadeIn` — maintains visual continuity.

### Color as Meaning
Consistent encoding throughout the video:
- **Input/data**: BLUE
- **Output/result**: GREEN
- **Current focus**: YELLOW
- **Error/negative**: RED
- **Neutral**: WHITE/GREY

### Spatial Layout
- Left → Right: time, transformation flow
- Center: main focus
- Top edge: titles/context
- Bottom: equations/formulas
- Corners: variable displays, legends

### 3b1b Color Palette

## Common Pitfalls

1. **Version confusion** — `from manim import *` is ManimCE; `from manimlib import *` is ManimGL
2. **Transform keeps old reference** — Use `ReplacementTransform` instead of `Transform` when you want the new variable
3. **AddTextLetterByLetter is Text-only** — Only works with `Text`, not `MathTex`
4. **LaTeX not raw** — Always `r"..."` for LaTeX strings
5. **fill_opacity defaults to 0** — Shapes are transparent by default, set `fill_opacity`
6. **Updaters persist** — Call `mob.clear_updaters()` when done; too many slow rendering
7. **always_redraw performance** — Recreates the mobject every frame; keep the lambda lightweight
8. **Cache staleness** — Use `manim --disable_caching` if animations seem stuck on old code
9. **manimpango issues** — If text rendering fails, ensure `manimpango` is installed and system fonts are available
10. **Graph edge keys** — `graph.edges` uses tuples as keys; for undirected graphs, `(u,v)` and `(v,u)` may differ — use the same order as construction
11. **Table indexing is 1-based** — `table.get_cell((1, 1))` is the first cell, not `(0, 0)`
12. **MoveToTarget requires generate_target()** — Call `mob.generate_target()` before modifying `mob.target`
13. **Restore requires save_state()** — Call `mob.save_state()` before using `Restore(mob)`
14. **Code param is `code_string`** — Not `code`. No `font_size` param; use `.scale()` instead. Access lines via `.code_lines[i]`
15. **Brace.get_text() doesn't accept font_size** — Create `Text()` separately and position with `.next_to(brace, direction)`
